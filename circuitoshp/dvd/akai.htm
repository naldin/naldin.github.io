AKAI DV-P1000 (Clone: Philco 1500)

Tomar muito cuidado ao acessar a área de ajustes deste player. Voce terá acesso a diversos ajustes que poderão levar o aparelho a um black-out permanente. Anote todos os dados antes de realizar as alterações.

Desligue o aparelho através da tecla do painel.Acione e segure as teclas 'Eject' e 'A-B', ao mesmo tempo, então acione a tecla 'on/off' para ligar o aparelho. Quando o texto 'A:00 D:12'(ou similar) aparecer no display, libere as duas teclas (Eject e A-B). Pressionando a tecla 'Play' para subir e a tecla 'Stop' para descer, mude o valor de 'A' até ler 'A:05' Pressionando a tecla 'Skip-Forward' para subir e a tecla 'Skip-Backward' para descer, mude o valor de 'D' até ler 'D:00' para codefree, 'D:01' para região 1 ou ' D:04' para região 4, etc. Após, pressione a tecla "Pause" para aceitar as alterações. Pressione a tecla 'on/off' para desligar o aparelho. Após alguns segundos, torne a ligar o aparelho, instale um disco e 'play'.


AKAI - DV-P2000

Este codigo NÃO funciona nos mais recentes aparelhos. A tecla preta não foi instalada!

Desligue o aparelho. Mantenha acionadas as teclas as teclas 'Fast Forward' e 'Stop' juntas com a tecla preta localizada à direita, no lado inferior do aparelho. Então, acione a tecla 'on/off' para ligar o aparelho. Quando aparecer o logo da Akai, libere as teclas 'Fast Forward' e 'Stop', mas mantenha a tecla preta acionada. Uma lista de numeros aparecerão na tela. Pressione uma vez a tecla 'Enter' no controle remoto. Isto permitirá selecionar a segunda coluna de números. Pressione o cursor 'direita' no controle remoto para selecionar o segundo número. Mude este valor para 00 para multi-região, 01 para Reg. 1, 02 para Reg. 2, para reg. 4, etc. Acione 'Enter' uma vez para confirmar a troca de região. Acione a tecla 'on/off' no player para desligar mais uma vez, e então libere a tecla preta. 8. Religue o aparelho, instale um disco e 'play'.

Esquema para fazer o cabo DAU-9P (MBUS)
 

Este cabo é feito para trabalhar com o Wintesla(M2Bus).

Se voce tiver problemas para encontrar o C.I MAX-232, vá na pagina: http://www.reidosom.com.br que sera facil de encontrar. Caso nao encontre o conector do telefone, procure em "camelos" um fone de ouvido para o telefone, e retire o conector, este fone não é muito caro.

Todo o circuito é alimentado pelos pinos 7(RTS) e 4(DTR).D1 e D2 é necessario para não haver retorno do circuito. A voltagem é regulada em 5V pelo regulador(7805). Os diodos Zener DZ1 e DZ2 são muitos importantes para a segurança de seu telefone, pois uma pequena alta acima dos 3,3V , pode danificar seu telefone. Normalmente os valores de funcionamento estão entre 2,7V a 3.3V , tensão com o telefone conectado. Portando os valores de DZ1 e DZ2 podem ser proximos de 2,7V a 3,3V. Este valor é o suficiente para o funcionamento e está abaixo do valor maximo de voltagem. O resistor R1 de 1K é necessario para reduzir a voltagem no pino 10 do C.I. . O diodo D3 é usado para interface MBUS no caso de uma interface FBUS, nao é necessario este diodo.

Lista de componentes:

D1,D2,D3 - 1N4148
R1 - Resistor 1k
C1,C2 - Capacitor Ceramico 100nF
C3,C4,C5,C6 - Capacitores Eletroliticos 1 microF
DZ1,DZ2 - Diodes Zenner 3V3 ou 2V7
C.I - MAX232 ou HIN232
Regulador de voltagem 7805
Conector DB9 femea

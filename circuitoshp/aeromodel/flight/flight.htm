Como modificar um transmissor Futaba R/C
para uso como um controlador PC compatível

Este artigo explica como converter um transmissor Futaba Conquest 4NBF ou um 4NBL Attack em um controlador de jogo(Joystick) PC compatível com aileron, profundor, leme, motor e ajuste de trim. O Controlador de Simulador de Vôo trabalhará com qualquer simulador de vôo que usa um joystick e se corretamente configurado, deve-se calibrar corretamente usando a calibração padrão no jogo. O Controlador de Simulador de Vôo usa potenciômetros de 500K (no Brasil é encontrado o de 470K) ou 1M, permitindo quase a mesma gama de resistência dos potenciômetros de um joystick padrão. Nenhuma providência especial é necessária.

Lojas de hobby vendem a caixa do 4NBF sem cristal, bateria ou antena. Muitas pessoas têm transmissores velhos que podem ser usados. Tenha em mente que esta é uma modificação permanente. Esta conversão provavelmente trabalhará com outros transmissores mas não foi testado. Alguns transmissores, como o Futaba séries de FG, usa potenciômetros separados para o eixo de stick e trim. Neste caso, o transmissor pode ser feito, mas o trim não estará disponível.

O computador no qual o simulador será ligado tem que ter uma porta de jogo (geralmente a porta de jogo vem acompanhando a placa de som). Algumas placas mais velhas têm dois conectores separados, um para cada stick. Vale lembrar que cada lado do Joystick(motor/leme e profundor/ailerom ) corresponde a uma porta de jogo.

Iniciando

O primeiro passo que deve ser dado é ler este artigo em sua totalidade antes de realizar qualquer trabalho. Embora este projeto não seja especialmente difícil, é necessário um pouco de experiência mecânica e elétrica, como saber soldar fios e usar de maneira correta as ferramentas manuais. Não está além das habilidades do modelista comum. Este sistema utiliza 5 volt para o seu funcionamento. O terra e a linha de +5 volt  nunca devem tocar um ao outro.  Deve ter cuidado para conectar tudo corretamente para não danificar a fonte do computador ou a placa com a porta de jogo. Embora as instruções e diagramas incluídos neste artigo foram conferidas, enganos acontecem. Toda a instalação elétrica deve ser conferida duas vezes antes de conectar qualquer coisa. Qualquer risco levado no uso deste artigo  é de responsabilidade do usuário.

O que é necessário

É necessária uma superfície limpa e organizada de forma que partes não serão perdidas. As ferramentas requeridas são um voltímetro ou provador de continuidade, um ferro de soldar com baixo poder de solda e alguma lixa fina. As partes requeridas são:

  • 4 (quatro) potenciômetros de 470 K LIN (470,000 Ohms),
  • um limpa contato (não WD-40 ou qualquer outro lubrificante),
  • espaguete termo retrátil ou um rolo de fita isolante

Para facilitar a montagem, os melhores potenciômetros para usar são potenciômetros de carbono com 5/8 " de diâmetro ou mini que estão disponíveis em muitas lojas de suprimentos de eletrônica. (* Experiência própria , desmonte os potenciômetros e troque apenas o contato (aquela parte preta interna)). É aconselhável adquirir cinco ou seis potenciômetros para o caso de ocorrer erros ao cortar os cabos. Finalmente, um cabo de condutor de 8 vias (*ou flat cable de impressora) e um conector macho de 15 pinos será requerido. Um interruptor de contato momentâneo será requerido se o joystick for usado com um simulador de vôo ou jogo que usam um "botão de Tiro".

Desmontando o Transmissor

Todos os componentes eletrônicos devem ser removidos do transmissor com exceção da chave trainner se estiver instalado. A antena, bateria, o conector do trainner, placa do circuito principal, etc. não são requeridos e devem ser removidos para reduzir a chance de curto-circuito entre a linha de +5 volt e o terra. Os fios para os quatro potenciômetros de cada conjunto são cortados. Os dois conjuntos são afastadas removendo os quatro parafusos cada um num lugar. Os conjuntos são diferentes e têm que voltar no mesmo local do qual eles foram levados. Os componentes removidos podem ser usados como peças sobressalentes para outro transmissor.

Os potenciômetros no rádio devem ser substituídos a fim de aumentar a gama de resistência para se assemelhar a um joystick comum. O conjunto do motor é sem dúvida o mais simples. A tira de aço do stick do motor deve ser afastada neste momento. Os potenciômetros são presos a base por uma rosca de metal mas são livres para virar, permitindo o ajuste do trim. Todas estas peças deveriam ser colocadas em uma sacola plástica assim não serão perdidas. Para remover o potenciômetro, são usados alicates para soltar com cuidado a rosca de metal da face interior do suporte. Deve ser tomado cuidado para evitar danos ao conjunto. Quando estiver solto, pode ser desparafusado à mão. As peças devem ser examinadas antes de desmontadas.

Modificando os Potenciômetros Novos

O eixo do potenciômetro é semi-circular com um entalhe em um lado e é mais curto que o do potenciômetro de substituição. Devem ser cortados para igualar aos potenciômetros velhos. Primeiro, o eixo do potenciômetro novo é cortado no mesmo comprimento do velho.

O próximo passo é um dos mais críticos no processo funcional do controlador. Os ajustes dos novos potenciômetros devem ser baseados no esquema abaixo onde os extremos de resistência dos potenciômetros modificados serão 5 Ohms e 100 K. Se a caixa que está sendo usada é diferente de um Futaba 4NBF ou 4NBL, uma modificação será exigida para determinar a posição correta e o método de fixação do cabo.

Note que todos os potenciômetros estão com o mesmo corte menos o do motor. O potenciômetro do motor está montado invertido, assim a orientação do entalhe é oposto que dos outros. O ângulo correto para o entalhe é determinado virando todo o eixo até o encosto na direção indicada acima. Um lápis ou marcador é usado para marcar onde o entalhe devera ser cortado. O ângulo do entalhe é bastante crítico mas este método renderá resultados satisfatórios se tiver cautela na localização do entalhe. O método usado para produzir o entalhe é dependente do equipamento utilizado. Embora o método, a face deve ser tão plana quanto possível e como próximo do ângulo correto. Deve ter cautela para impedir os limalhas de metal dentro dos potenciômetros. Depois que o entalhe seja cortado, a lixa fina é usada para remover quaisquer irregularidades do corte.

Limpar os contatos do potenciômetro para remover qualquer pó de metal que pode ter entrado. O pino de alinhamento no caso do potenciômetro novo pode não se alinhar com o buraco do antigo. Todas as peças devem ser alinhadas da mesma maneira como o original. Os potenciômetros novos podem necessitar mais tensão que os originais. Assegure que o stick mova livremente e se auto centralize quando liberado.

Montagem

Neste momento, o trabalho duro terminou mas tendo o mesmo cuidado nas modificações mecânicas deve ser levado na instalação elétrica. Um engano pode resultar neste momento em dano à fonte do computador ou a placa com a porta de jogo. O cabo do controle tem que estar usando um conector macho de 15 pinos em uma extremidade e fios na outra. Um voltímetro é usado para localizar os fios fora para o pino correspondente do conector. O conector deve ter números muito pequenos impressos adjacente a cada pino. Se não for feito isso, o diagrama abaixo pode ser usado para determinar o número do pino.

Depois que os fios corretos são determinados,  a instalação elétrica é feita. A primeira coisa que deve ser feita é isolar o terra (pino 5) e fios de +5 volt (pinos 8, 9 e 15). Cuidado com estes fios pois os mesmos podem gerar um curto-circuito. O curto-circuito entre o terra e a linha de +5 volts pode queimar a fonte do computador. A extremidade do fio do cabo pode ser inserido pelo buraco de antena ou qualquer outro ponto que pareça conveniente. O uso do interruptor do botão trainner  como um botão de tiro ou trem retrátil é opcional. Há três fios no interruptor de treinador. O voltímetro é usado para encontrar os dois fios que têm continuidade quando o interruptor de botão trainner é apertado e o outro fio é afastado. O terra (pino 4) é conectado a um dos fios restantes do botão trainner e o fio do pino 2 no conector é conectado ao outro. Estas conexões devem ser soldadas para evitar o toque de um fio de +5 volt. A linha de +5 volt será conectada aos potenciômetros assim devem ser levados esforços para assegurar que nenhum fio terra possa estabelecer contato com um potenciômetro. Recorrendo ao diagrama abaixo, os +5 volts serão conectados para contatar C em cada potenciômetro menos o motor que conectou para contatar o A. Então cada fio de eixo é conectado para contatar B do potenciômetro apropriado. O pino 3 no conector vai contatar B no potenciômetro de aileron, etc. Quando este passo é completado, haverá fios deixados para outras funções. Um interruptor de contato momentâneo pode ser conectado a estes para acrescentar funcionalidade ao controlador. Quaisquer fios que não forem usados devem ser removidos ou  isolados.

Testando o Controlador

Não conecte o controlador ao computador ainda. Ligue o computador e deixe inicializar no DOS. Cautelosamente, conecte o controlador à porta de jogo enquanto observa o monitor. Se na tela aparecer algo branco, desconecte o controlador imediatamente. Esta é uma indicação que há um curto-circuito. Em algum lugar na caixa, há um fio terra que toca um fio de +5 volt. Um pequeno curto não deveria danificar nada e o computador deveria reiniciar. Abra a caixa e cheque tudo novamente. Esta condição deve ser corrigida antes que o controlador seja conectado novamente.

Se tudo parecer normal quando o controlador estiver fechado, está na hora de testá-lo. O controlador deverá ser testado primeiro com o programa de JOYTEST de Dave Brown Products. Este programa confirmará que estão funcionando os potenciômetros e botões corretamente e que os valores de resistência exigidos podem ser alcançados. Quando os sticks são movidos em qualquer direção, o indicador correspondente tem que se mudar para a mesma direção e quando os sticks são movidos aos limites, os indicadores têm que mover aos limites. Como é apertado cada botão, um ícone mudará a aparência para indicar que foi apertado. Se um indicador de stick mover na direção oposta, então o potenciômetro foi montado invertido. O potenciômetro deve ser refeito antes de proceder ao passo de calibração.

Figura do programa JOYTEST

Se houver uma ferramenta de calibração de joystick com a placa de porta de jogo, use isto para determinar se o controlador trabalha corretamente. Se no computador estiver processando Microsoft Windows 95/98, use o ícone de Calibração de Joystick no painel de controle. Se nenhum destes forem disponíveis, o controlador pode ser calibrado dentro de um jogo que mostra as posições do stick depois de calibrado, como Flight Unlimited, Flight Simulator ou Real Flight.  Siga as instruções do software para ajustar os trims para o centro, e confirme que há uma resposta linear em cada direção do eixo de cada stick. Se há uma direção de um eixo que tem resolução melhor que outro, tente virar o controle trim para aquele eixo na direção ruim e recalibre. Se os ajustes dos sticks estão fora muito distante de posição, pode ser necessário abrir a caixa e virar o potenciômetro inteiro. Assegure que os sticks voltem para o centro e que há nenhum stick travado em uma posição diferente do centro.

Pronto Para Uso

O design original deste controlador veio de Robert Osorio. Embora seja principalmente planejado para uso com simuladores de vôo, trabalha bem com outros jogos que beneficiam controle preciso.

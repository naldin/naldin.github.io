Notas gerais sobre as interfaces

Todas as interfaces desta pagina são para serem usadas somente com transmissores PPM. Elas não suportam nenhum transmissor PCM.
Todos os transistores usados são de uso geral, e podem ser substituidos, os transistores são usados com ganho 100 ou maior.
Os CIs PIC usados, sao microcontroladores programaveis, e necessitam ser programados para funcionarem corretamente.
As interfaces mais simples serial ou paralela, não trabalham com WinNT ou WinXP.
A interface PIC serial trabalha somente com o simulador FMS.
A interface Joystick pode trabalhar com qualquer simulador ou game.
As interfaces PIC e Joystick foram testadas e trabalharam corretamente com os seguintes transmissores: JR Quattro, JR XF421, JR XP652, JR XP662, JR 10X, Futaba FF6, Sanwa 400, Hytec Flash 4, Hitec Focus 4, Hitec Focus 6.

 

Interface com o PIC 16F873 para a porta do Joystick e no FMS para serial

Versão: 1.05

Esta interface trabalha com qualquer simulador/game, e pode trabalhar em qualquer sistema operando pela porta do joystick. A interface serial funciona somente com o FMS.

Caracteristicas:

Interface do Joystick
Canais 1-4 do transmissor são usados pelos eixos 1-4.
Canais 5-8 do transmissor são usados para os botões 1-4.
Interface pode trabalhar com qualquer game/simulador que suporte joystick.
Usa simples 4 canais do potenciometro digital AD5204B100.
Pode ser usado com conversores "joystick para USB".
Interface Serial PIC
Selecionavel entre 9600 e 19200 baud por meio de um botão.
Os canais 1-8 do radio suportam trabalhar com 19200 baud.
Trabalha somente com o FMS.
Setup simples por meio de um botão
Com nenhum sinal PPM presente, o botão trava entre 9600 & 19200 baud.
Com sinal PPM presente, salva a posição do controle.
Dados são atualizados a cada 22ms.
Não necessita calibrar.
Desenvolvido para PC.
Pode ser conectado no Joystick e na porta serial ao mesmo tempo.
Quatro "botoes de fogo" para transmissores com menos de 5 canais.
Led indicador de sinal presente.
Poderá trabalhar com qualquer canal de 2-8 do transmissor.
Livre para todos, incluindo o codigo fonte.
Voce pode precisar de assistencia para a mudança do codigo para outro tipo de potenciometros digital ou protocolo serial. , Eu ficarei contente em ajudar na mudança do codigo do PIC.

Interface com o PIC 12C508 para a porta do Joystick

Versão: 1.01

Esta é a versão mais barata para a porta do joystick, ela usa um PIC 12C508 e um potenciometro digital Ad5204B100. Temos aqui o esquema, a placa de circuito impresso(PCB) e o codigo fonte em HEX. 

Temos o codigo em HEX para as versões 12C508, 12C508A, 12C509, 12C509A.

A interface pode trabalhar com qualquer simulador/games que funcione pela porta do joystick , e opera em qualquer sistema, incluindo o DOS. 

Parametros de programação do PIC
Oscilador: RC Interno
Whatchdog Timer: On
Master Clear: Interno

Este circuito foi testado e trabalhou corretamente com transmissores JR.
